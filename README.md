# libinterpret_hse

This library provides an interpreter for converting from an hse abstract syntax tree to an hse graph.

Dependencies:
 * parse_hse
	 - parse_boolean
		 - parse
			 - common
 * hse
	 - boolean
		 - common
 * interpret_boolean
	 - parse_boolean
		 - parse
			 - common
	 - boolean
		 - common
